HTML
HyperText Markup Language
El lenguaje que da estructura y vida a cada página web que existe en internet.
01

¿Qué es HTML?

HTML es el lenguaje de marcado estándar para crear páginas web. Usa etiquetas para definir la estructura del contenido: títulos, párrafos, imágenes, enlaces y más.

02

Por qué es importante

  • Base de toda la web — sin HTML no existen sitios web
  • Trabaja junto con CSS (estilos) y JavaScript (lógica)
  • Primer paso para aprender desarrollo web
  • Usado por millones de desarrolladores en todo el mundo
03

Estructura básica

<!DOCTYPE html> <html lang="es"> <head> <title>Mi Página</title> </head> <body> <h1>¡Hola Mundo!</h1> </body> </html>
04

Etiquetas esenciales

<h1>Título
<p>Párrafo
<a>Enlace
<img>Imagen
<div>Contenedor
<ul>Lista
<form>Formulario
<nav>Navegación
