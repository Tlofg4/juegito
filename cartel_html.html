HTML

HyperText Markup Language

📝 ¿Qué es HTML?

  • Lenguaje de marcado estándar
  • Crea la estructura de páginas web
  • Usa etiquetas para definir elementos

🏗️ Estructura Básica

<!DOCTYPE html> <html lang="es"> <head> <title>Mi Página</title> </head> <body> <h1>¡Hola Mundo!</h1> </body> </html>

🔧 Etiquetas Principales

  • <h1> - <h6> Encabezados
  • <p> Párrafos
  • <a> Enlaces
  • <img> Imágenes
  • <div> Contenedores
